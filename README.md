Features

Dynamic Light & Dark Mode: A seamless toggle between a dark, sci-fi slate theme and a clean, accessible light theme.
Glassmorphic UI/UX: Styled using modern CSS variables, backdrops, and blur effects for a premium look and feel.
Fully Responsive & Mobile-First: Optimized for all screen sizes, from ultra-wide desktops to smartphones.
Bug-Free Components:Sidebar Navigation: Fixed mobile overlay and sidebar visibility to prevent ghosting and overlapping.
Certification Grid: Responsive layout with robust padding rules preventing absolute-positioned "View Certificate" badges from overlapping long titles.
Functional Contact Form: Built-in validation, spam protection, and automated routing via FormSubmit.
Super Lightweight: Built with pure vanilla HTML5, CSS3, and JavaScript—no heavy frameworks, ensuring lightning-fast load times.
Tech Stack

HTML5 - Semantic structure
CSS3 - Custom variables, Flexbox, CSS Grid, Glassmorphic effects, responsive media queries
JavaScript (ES6) - Theme switching, sidebar toggles, scroll behaviors
Icons - Custom SVGs, SimpleIcons
