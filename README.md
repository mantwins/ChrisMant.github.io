## Features

- Dynamic Light & Dark Mode: A seamless toggle between a dark slate theme and a clean, accessible light theme.
- Glassmorphic UI/UX: Styled using modern CSS custom properties, backdrop filters, and blur effects for a clean visual hierarchy.
- Fully Responsive & Mobile-First: Optimized across screen sizes, from mobile devices to desktop displays.
- Refined Components:

  - Sidebar Navigation: Mobile overlay and slide-out navigation menu for intuitive site navigation.

  - Projects & Certifications Grids: Structured layout with hover dynamics and direct credential/repository links.

  - Functional Contact Form: Asynchronous submission support via FormSubmit with integrated toast feedback.

- Lightweight Architecture: Built with pure vanilla HTML5, CSS3, and JavaScript with zero external runtime framework dependencies.
